# vue-jqueryui

## 项目说明
```
这是一个 vue 使用 jQueryUI 的小栗子
业务场景：
1. 将右侧的CPL数据拖拽到"广告计划编辑区"，默认为第一条
2. "广告计划编辑区"的数据可以拖拽到左右两边删除，也可以拖动进行排序
3. "广告计划编辑区"的数据被改动后，可以点击"恢复"按钮回到数据初始状态

可以执行下面的命令查看效果
```
